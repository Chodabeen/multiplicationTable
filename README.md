# multiplicationTable

# 홀수단 또는 짝수단의 구구단의 열의 개수를 맞추어 출력하기

출력모드(홀수 또는 짝수)와 열 갯수를 입력받아 이에 따라 홀수 단 또는 짝수 단의 구구단만을 1줄에 열 개수만큼씩 출력

단, 출력모드 입력 내용이 1이면 홀수단, 2이면 짝수단으로 결정하도록 한다

변수는 다음과 같이 사용 

int mode; // 출력모드(1: 홀수단, 2: 짝수단) 

int column; // 열 개수 

int i, j; // 반복문 사용을 위한 변수
